# Simple VR Seated Camera Unlocker

This mod unlocks the horizontal camera rotation (right thumbstick) while the player is seated in Skyrim VR.
It allows you to look around freely without twisting your real-world body.

> このMODは、Skyrim VRで座っている状態（着席モード）での**水平方向のカメラ回転（右スティック）**を解除・有効化します。
> 現実の体を無理にひねることなく、自由に周囲を見渡せるようになります。

**Note:** Vertical look (up/down) is still controlled by your HMD movement due to game engine limitations.

> **注意:** ゲームエンジンの制限により、**上下（垂直方向）の視点移動**は引き続きHMD（ヘッドセット）の動きで操作する必要があります。

## Download & Usage (ダウンロードと使い方)
Please check the "Releases" on the right side of this page to download the latest zip file.

> このページの右側にある「Releases」から、最新のzipファイルをダウンロードしてください。

Import the downloaded zip file into your mod manager, or place the contents of the zip file into the SkyrimVR Data folder.

> ダウンロードしたZipを、MOD管理ツールで取り込むか、Zipファイルの中身をSkyrimVRのDataフォルダ内に配置してください。
